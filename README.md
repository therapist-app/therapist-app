# Therapist App
## Frontend

### How to run

1. Install nodejs 22.14.0 -> when typing `node -v` in the terminal, it should show `v22.14.0`
2. `cd frontend`
3. `npm install`
4. `npm run dev`
5. Visit `http://localhost:5173` in your browser

## Backend

### How to run

1. Install openjdk-21: `java --version`should output `openjdk 21 2023-09-19`
2. `cd backend`
3. `./gradlew bootRun`
